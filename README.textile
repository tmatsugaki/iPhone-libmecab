h1. 「形態素解析エンジン Mecab on iPhone のサンプルアプリケーション」の機能拡張版です。

中学生でも分かる一般的な出力形式（※）を追加しました。

その他、以下の特徴があります。
・解析した文章は <b>iTunesファイル共有</b>で読み書き可能なファイルに逐次、追加・保存されるので<u>再利用</u>が可能。
・起動時の解析結果は簡易表示ですが、<b>シングルタップ</b>（個別に）・<b>長押し</b>（纏めて）<u>表示モード</u>（簡易／詳細）を切り替えることが可能。
・解析結果のリストのセルを<b>ダブルタップ</b>すると、当該の<u>品詞の wiki ページ</u>を表示する。

※）「<b>学校文法</b>」ボタンが ON の場合は、<font color="blue">文節毎に色</font>が付きます。不適切な文節があった場合は対応しますので、ご連絡ください。
